# Trabalho-Pratico - A3
Trabalho Prático - Modelos, Métodos e Técnicas da Engenharia de Softwa

## Objetivo do Trabalho

O objetivo deste trabalho é aplicar, de forma prática, os conceitos estudados na UC de Engenharia de Software (prática e teoria) por meio da análise, modelagem e desenvolvimento de uma solução de software baseada em um problema real.

Os alunos deverão identificar um problema concreto existente em seu contexto social, acadêmico ou profissional, realizar o levantamento e análise dos requisitos envolvidos e desenvolver uma solução back-end em Java para resolver parte desse problema.

O foco do trabalho será o desenvolvimento de uma parte relevante da solução, aplicando conceitos de Engenharia de Software, orientação a objetos, modelagem, testes e padrões de projeto.

### Definição do problema
Muitas clínicas populares e pequenos consultórios ainda realizam o agendamento de consultas de forma manual, utilizando agendas físicas ou registros em papel. Esse processo gera diversos problemas, como:
- Dificuldade em controlar horários disponíveis.
- Risco de marcação duplicada.
- Perda de informações importantes sobre pacientes e consultas.
- Pouca flexibilidade para reagendamentos ou cancelamentos.

 ### Usuários envolvidos
 - Recepcionista/Administrador: responsável por cadastrar pacientes, médicos e organizar a agenda.
 - Paciente: precisa marcar, cancelar ou consultar seus horários de atendimento.

### Solução proposta
Será desenvolvido um sistema back-end em Java para apoiar o processo de agendamento de consultas. O sistema permitirá:
1. Cadastro de pacientes.
2. Cadastro de médicos.
3. Agendamento de consultas com controle de horários disponíveis.
4. Cancelamento de consultas.
5. Consulta da agenda de atendimentos.

### Escopo delimitado
O projeto terá foco no back-end, com regras de negócio bem definidas, aplicação dos princípios SOLID, uso de padrões de projeto e testes unitários. Não será desenvolvida interface gráfica completa, mas sim uma solução funcional que demonstre a aplicação prática dos conceitos de Engenharia de Software.
